# 오픈하우스 점수판 시스템

교내 동아리연합회의 '오픈하우스' 행사에서 사용되는 점수판 시스템의 프론트엔드 애플리케이션입니다.

## 기술 스택

- React 18
- TypeScript
- React Router DOM
- Vite

## 주요 기능

1. **점수판 리스트 화면**: 등록된 모든 점수판을 확인하고 접근할 수 있습니다.
2. **로그인 화면**: 관리자 계정으로 로그인할 수 있습니다.
3. **관리자 초기화면**: 생성된 점수판 목록과 관리 기능에 접근할 수 있습니다.
4. **점수 관리 페이지**: 각 팀의 점수를 증감시키고 히스토리를 관리할 수 있습니다.
5. **점수판 설정 화면**: 점수판의 이름, 공지사항, 경기 시간, 참여자 등을 설정할 수 있습니다.
6. **상세 점수판 페이지**: 실시간으로 점수와 순위를 확인할 수 있는 공개 화면입니다.

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 데이터 저장

현재는 브라우저의 LocalStorage를 사용하여 데이터를 저장합니다. 실제 운영 환경에서는 백엔드 서버와 연동하여 데이터를 관리해야 합니다.

## 프로젝트 구조

```
src/
├── components/      # 재사용 가능한 컴포넌트
├── contexts/        # Context API (상태 관리)
├── pages/          # 페이지 컴포넌트
├── types/          # TypeScript 타입 정의
├── utils/          # 유틸리티 함수
├── App.tsx         # 메인 App 컴포넌트
└── main.tsx        # 진입점
```
